Lango (2018)

Homepage of Lango

Lango is a flashcard-based language learning website built using NodeJS, React and SQLite. Users can create an account, create flashcards and practice! The website is supported by Google Translate and is available for those who just want a quick start. Or, you can create cards by hand and review them later.

Chatbox (2019)

A Chatbox poll

Chatbox is a poll/forum generator built using .NET MVC and React. Users are greeted to a simple homepage where they can enter a discussion topic, and instantly, a URL is created for them to share with others. The discussion page is powered by Azure SignalR to send messages to everyone on the same URL, and SQLExpress, to store comments and discussion topics.
